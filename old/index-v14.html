DEVICES DISCOVER EACH OTHER · NO PERSISTENT SERVER REQUIRED

🔒

Decentralized relay, no server bottleneck

Devices discover each other and relay tasks directly. No persistent server in the middle. Scales with users, not infrastructure.

🖥

Hardware-based device identity

Each device is identified by its hardware UUID, not a session token. Survives OS reinstalls. Your device is your identity.

⚡

Serialized task queue with deduplication

One task at a time per device. Atomic status transitions. No race conditions when agents act on real things.

🧰

Native instance, full local privileges

Not sandboxed VMs or cloud screenshots. Each Orion runs natively with real access to screen, files, browser, and voice.
