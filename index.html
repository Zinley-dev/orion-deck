02 / 12

Problem

Today’s personal agents
are stranded.

Stuck in a cloud sandbox or a dead gadget. No identity, no continuity. More an internal prompt tool than something that can actually represent a person.

01 · Stuck in a box

The agent lives in one silo. The user lives everywhere.

Manus runs in a cloud VM. Perplexity orchestrates in the cloud. Zo runs on its own server. OpenClaw lives in a chat thread. The user’s real context, on phone, laptop, calendar, inbox, sits untouched.

02 · Hardware can’t fix it

A new gadget can’t reach the rest.

Humane: $230M raised, sold to HP for $116M, bricked the AI Pin in Feb 2025. Rabbit R1: 100K pre-orders, 5K daily users 5 months in. Same gap, new shape.

03 · No identity, no trust

If it can’t be reached, it can’t represent.

OpenClaw can place calls through MCP. But the number is anonymous and the handle isn’t tied to a person. No reason for the other side to pick up, no way to recognize the line later.

04 · Every agent starts cold

Tell one. The next has no idea.

Say “join that meeting” and one agent dials in through a voice API. It doesn’t know which meeting, who’s on it, what you’d say, or how to follow up after.

03 / 12

Solution

We call it the
personal representative.

Four properties working together.

01 · Personal entity

Its own number, email, and handle.

A representative tied to the owner. The owner sends it out for agentic voicemail, calls, meetings, and outreach on their behalf.

02 · Continuity

Every interface remembers.

Web, desktop, mobile, call, text, email. Start on one surface, continue on any other. The thread carries over.

03 · Interconnected

Travels across the owner’s devices.

One representative across the devices the owner gives it permission to use. Picks up context wherever it left off.

04 · People memory

Memory of the people the owner talks to.

Built up over time from calls, replies, and intros. Names, history, what each person tends to ask.

Personal, continuous, interconnected.

How it works · sample flow

Agentic voicemail,
end to end across the owner’s devices.

One example flow. The same setup handles other tasks the owner gives the representative.

01

Install on any device the owner uses.

Phone, laptop, web. Each install becomes a place the representative can run from.

02

Buy a phone number from Orion. Optional

If the owner wants a real-world identity for the rep. A real number, provisioned to the agent. Available in the US and Canada.

03

Calls route to the agent when the owner is busy.

The agent picks up using the owner’s pre-set instructions, asks follow-ups, takes notes, and handles bookings.

04

Recap by email. Reply to continue.

The agent emails the owner a recap of what it did. The owner replies in that thread to keep the conversation going.

Early use cases

What people use it for
today.

Two early use cases.

01 · Daily work

Same kinds of tasks people use Claude Cowork for.

Drafting, scheduling, research, follow-ups. Orion can also pull context from the owner’s phone, laptop, and web in the same session.

02 · Agentic voicemail

Calls route to Orion when the owner is busy.

Not a recording. The call connects to the representative. Orion answers using the owner’s pre-set instructions and notifies the owner afterward. Each call adds to the representative’s memory of the caller: name, history, what they tend to ask.

08 / 12

Why now

The race is happening
in 2026.

Adoption is moving fast, every major lab has shipped an agent, and the personal-entity layer is still open.

01 · Adoption

~500M people use AI daily.

35% of consumers globally interact with an AI agent or assistant daily. 44% of US consumers say they’d use one as a personal assistant, rising to 70% of Gen Z. The agentic AI market is on track from $7.4B in 2025 to $100B+ by 2032.

02 · The race

Every major lab has shipped this year.

ChatGPT Atlas, Perplexity Comet (free across platforms in March 2026), Manus (acquired by Meta), OpenClaw (3.2M users, 346K GitHub stars), Claude for Chrome, Google Project Mariner. All launched or accelerated in the last 12 months.

03 · The gap

Nobody ships a personal entity yet.

Every entrant lives inside a browser tab, a chat window, or a cloud VM. None come with their own number, memory, and presence across the user’s devices. That layer is still open.
